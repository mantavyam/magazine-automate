## Most Used Keyboard Shortcuts:

### WINDOWS

Win + V = Clipboard&#x20;

Fn + F11 = Full Screen Mode

{% hint style="info" %}
Where Win means Windows Key

Where 'Fn' means function key located on the left of windows key.

On PC Keyboard, directly press F11 (No need of Fn Key)
{% endhint %}
